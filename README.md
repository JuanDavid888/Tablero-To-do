# **Tablero To-Do**

Este proyecto es un pequeño tablero Kanban hecho completamente con HTML, CSS y JavaScript Vanilla, sin frameworks.
La idea es tener una herramienta sencilla para organizar tareas en tres columnas:

- Hacer

- Haciendo

- Hecho

## **Funcionalidades principales**

Agregar tareas desde un formulario con:

- Nombre

- Descripción

- Prioridad (1-10)

- Estimación en horas (opcional)

- Responsables (opcional)


## **Estructura del proyecto**

```
tablero_to_do/
│
├── index.html        → Estructura principal del tablero
│
├── css/
│   └── styles.css    → Estilos del tablero y colores de las columnas
│
└── js/
    └── main.js       → Lógica: crear, editar, mover y guardar tareas
```

## **¿Cómo usarlo?**

**1.** Abre el archivo index.html en tu navegador.

**2.** Presiona el botón “Agregar Tarea”.

**3.** Completa los campos del formulario.

**4.** Arrastra las tareas entre columnas según avancen.

**5.** Tus cambios se guardan automáticamente.
